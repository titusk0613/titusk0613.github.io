HTML Note 3

Images

<img>

  • Defines an image in a HTML page

Attribute:

src

  • Specifies the URL of an image

Using URL of image - Method 1

  • <img src=" Copy URL adress

Saving the image file - Method 2

  • First, the image should be saved in the folder containing the HTML code - Then add it in using "image/"

Attribute:

Height and Width

  • <img src=" Copy URL adress height="300" width="300"
  • Used to resize the image in pixels
  • You can specify only one height or width and the aspect ratio will be adjusted
